# Korean_pronunciation_analysis
외국인의 한국어 발음 오류 패턴 비교 분석입니다.
외국인 학습자들의 한국어 발화 데이터셋을 이용하여 권역별(국가별) 발음 오류 패턴을 비교 분석하는 task 입니다.
Chinese, English, Spanish 의 L1 외국인들의 한국어 발음 오류 패턴을 분석하였습니다.
